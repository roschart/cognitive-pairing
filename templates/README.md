# Templates

Ready-to-copy templates for each artifact type.

| Template                                        | Use for                          |
|-------------------------------------------------|----------------------------------|
| [plan.md](plan.md)                             | Living plan (any complexity)     |
| [checkpoint.md](checkpoint.md)                 | Stable state checkpoint          |
| [memory.md](memory.md)                         | Working memory (active.md)       |
| [session-summary.md](session-summary.md)       | Session delta (cp-session-end)   |
| [hydration-prompt.md](hydration-prompt.md)     | Session hydration prompt         |
| [decision.md](decision.md)                     | ADR-style decision record        |

Copy the relevant template to your project directory and populate it.
Never commit a template with placeholder text still in it.
